# 100 Days of Code - Python Projects

A collection of Python projects built during the 100 Days of Code course by Dr. Angela Yu.
Each project covers different aspects of Python development — from command line scripts
to full stack web applications.

## 🚀 Projects

### Project 1 — Portfolio Website
A personal portfolio website built with Flask and Bootstrap.
Showcases projects, skills, and contact information.
**Tech:** Python, Flask, HTML, CSS, Bootstrap

---

### Project 2 — Morse Code Converter
Command line application that converts any text input into Morse code.
Handles uppercase/lowercase, numbers, punctuation and unknown characters.
**Tech:** Python, Dictionary, String manipulation

---

### Project 3 — Tic Tac Toe
Two player command line Tic Tac Toe game with input validation,
win detection across all 8 combinations and draw detection.
**Tech:** Python, Lists, Functions, Game logic

---

### Project 4 — Image Watermarker
Desktop GUI application that uploads an image and applies
text or logo watermarks with transparency support.
**Tech:** Python, Tkinter, Pillow, File dialogs

---

### Project 5 — Typing Speed Test
Desktop application that tests typing speed in words per minute.
Features countdown timer, word comparison, accuracy calculation and restart functionality.
**Tech:** Python, Tkinter, Time module

---

### Project 6 — Breakout Game
Classic Breakout arcade game with ball physics, paddle control,
brick collision detection, score tracking and level progression.
**Tech:** Python, Tkinter Canvas, OOP, Collision detection

---

### Project 7 — Cafe & Wifi Website
Full stack website displaying cafe data from a SQLite database.
Features add, delete, update and search functionality.
**Tech:** Python, Flask, SQLAlchemy, WTForms, Bootstrap

---

### Project 8 — Kanban Task Manager
Trello-style task management website with three columns —
Todo, In Progress and Done. Tasks can be moved, edited and deleted.
**Tech:** Python, Flask, SQLAlchemy, Bootstrap, WTForms

---

### Project 9 — Dangerous Writer
Desktop application inspired by The Most Dangerous Writing App.
Deletes all text if the user stops typing for more than 5 seconds.
Features custom timer selection, word count, sound alerts and file saving.
**Tech:** Python, Tkinter, winsound, File dialogs

---

### Project 10 — Color Extractor
Flask web application that uploads an image and extracts
the top 10 most common colors displayed as swatches with HEX codes.
**Tech:** Python, Flask, Pillow, NumPy, Collections, Bootstrap

---

### Project 11 — Book Scraper
Selenium web scraper that extracts book data from books.toscrape.com
across 10 pages and saves title, price, rating and availability to CSV.
**Tech:** Python, Selenium, CSV, Web scraping

---

### Project 12 — Dino Bot
PyAutoGUI bot that automatically plays the Chrome dinosaur game
by detecting obstacle pixel colors and pressing spacebar to jump.
**Tech:** Python, PyAutoGUI, Pillow, Screen automation

---

### Project 13 — Space Invaders
Classic Space Invaders arcade game with alien formation movement,
player and alien shooting, lives system, score tracking and level progression.
**Tech:** Python, Turtle graphics, OOP, Multiple files

---

### Project 14 — MealDB Recipe Website
Flask website powered by TheMealDB API.
Search meals by name, browse by category, view full recipes with
ingredients, instructions and YouTube links.
**Tech:** Python, Flask, REST API, Bootstrap, Jinja2

---

### Project 15 — PDF to Audiobook
Python script that converts PDF files to speech using pyttsx3.
Reads page by page with progress tracking and saves as WAV file.
**Tech:** Python, pypdf, pyttsx3, Tkinter file dialog

---

### Project 16 — Police Shootings Analysis
Flask data analysis website using Washington Post fatal police
shootings database and US Census data. Interactive charts covering
race, state, trends and demographics analysis.
**Tech:** Python, Flask, Pandas, Plotly, NumPy, Data analysis

---

### Project 17 — Laminex Document Scraper
Selenium scraper that extracts all technical documents from the
Laminex document library across 24 pages — capturing document name,
brand, type and issue date — saved to Excel.
**Tech:** Python, Selenium, openpyxl, Web scraping

---

### Project 18 — PDF Date Extractor
Desktop tool that scans a folder of PDF files and extracts
document name, issue date, expiry date and version number
using regex pattern matching. Saves results to Excel.
**Tech:** Python, pypdf, regex, openpyxl, Tkinter
